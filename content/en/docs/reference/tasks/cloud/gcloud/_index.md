title: Gcloud
description: Create and Manage Google Virtual Machines
---

## Synopsis

```text
Usage:
  gcloud vm-list
  gcloud vm-create <name>
  gcloud vm-delete <name>
  gcloud vm-getip  <name>
```

## Commands

```
  vm-list     lists the vm and their ips
  vm-create   create a vm
  vm-getip    get ip
  vm-delete   delete the vm
```
